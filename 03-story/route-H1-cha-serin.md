## Ch.6 "차세린 분기" (2026.06.01~07, 약 20분)

### 진입 조건
- `main_heroine === "H1"` (호감도 1위)

### 무대
- 의대 축제장 (낮), 동산병원 의국, 캠퍼스 야경 (밤)
- 배경: `bg_festival` (variant: day), `bg_dongsan_hallway`, `bg_campus_night_blossom` (variant: post_blossom)

### 비트
1. **축제 첫날 부스 운영** — 본과1 부스, 사람 많음
2. **차세린 깜짝 방문** — 후배 응원 + 견학 인연으로 부스 들름
3. **이벤트 CG: cg_serin_festival** — 사복(베이지 셔츠 원피스), 어색한 미소
4. **부스 정리 후 단둘이 캠퍼스 산책** — 벚꽃 진 후 신록의 캠퍼스 야경
5. **차세린의 고백 직전 망설임** — 전공의의 격무 + 7살 연하 의대생, "괜찮을까요"
6. **키 선택지 (Ch.6 키)** — 어떻게 답할지

### 키 선택지 (트루 트리거 키 #3)
- A. "**누나**" 처음 호칭 → +10 (트루 키)
- B. "선생님이라고 부르는 게 편해요" → +1
- C. "선배라고 부를게요" → +3

### 분기 평가 (Ch.6 끝)

```
key_count = countKeys(state, "H1")  // 0~3
aff = state.flags.H1

if aff >= 80 && key_count >= 3 → END_H1_TRUE
elif aff >= 60 && key_count >= 2 → END_H1_HAPPY
elif aff >= 40 → END_H1_NORMAL
else → END_H1_BAD
```
