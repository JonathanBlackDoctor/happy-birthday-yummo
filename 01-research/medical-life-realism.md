## 1. 본과 1학년 1학기 일상

### 1.1 첫 주 (OT)
- [ ] 본과 진학 직후 적응 분위기
- [ ] 예과 동기들과의 재회
- [ ] 시간표 빡빡함 (예과 대비)

### 1.2 해부학 실습 (Ch.2 핵심)
- [ ] 카데바 첫 대면 — **엄숙한 분위기**, 묵념, 대상자에게 감사
- [ ] 마스크 + 가운 + 앞치마 풀세팅
- [ ] 첫날 충격으로 토하는 학생 일반적
- [ ] 조원 5명 단위 (보통)
- [ ] 시간: 보통 오후, 2~3시간씩
- [ ] 학기 내내 진행 (3~6월)
- [ ] 시험: 실습 시험 + 필기

⚠️ **게임 표현 가드레일**: 카데바 직접 시각화 X. 실습실 입구, 가운 입은 모습, 마스크 쓴 모습까지만.

### 1.3 본과 시험 (Ch.4~5 핵심)
- [ ] 족보 문화 (선배 → 후배 시험 자료)
- [ ] 도서관 밤샘 (24시간 운영)
- [ ] 카페인 의존 (커피/에너지 드링크)
- [ ] 시험 직후 회식/멘붕 패턴

### 1.4 동산병원 견학 (Ch.3)
- [ ] 본과1이 받는 견학 코스
- [ ] 의국 분위기 (전공의 2~3년차들 분주)
- [ ] 펠로우/교수 만남

### 1.5 회식 / MT (Ch.5 핵심)
- [ ] 의대 회식 빈도 (분석상 술 853회 언급 → 매우 빈번)
- [ ] MT 시기 (보통 5월)
- [ ] 의대 특유 풍경 (술 강요 X 트렌드, 그러나 분위기 농후)

### 1.6 축제 / 체전 (Ch.6)
- [ ] 의대 축제 시기/규모
- [ ] 부스 운영
- [ ] 체전 (의대 vs 의대 또는 학과 간)
