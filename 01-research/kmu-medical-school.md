# 01-research/kmu-medical-school.md

> Claude Code 작업 지시: https://www.kmu-med.ac.kr/index.php 와 관련 자료를 수집하여
> 다음 항목을 채운다. 게임 내 디테일로 자연스럽게 사용될 정보만 추출 (학사 행정 X, 분위기/장소/이벤트 위주).

## 1. 캠퍼스 (대구 달서구 성서)

- [ ] 본관 외경 묘사 (이미지 프롬프트용)
- [ ] 강의실 분위기
- [ ] 도서관 위치/분위기
- [ ] 해부학 실습실 (외부 공개 정보 한정)
- [ ] 학생식당, 카페테리아
- [ ] 캠퍼스 주변 (성서 지역 카페, 식당, 술집)

## 2. 본과 1학년 커리큘럼 (2026년 기준 추정)

- [ ] 1학기 주요 과목 (해부학, 생화학, 생리학 등)
- [ ] 시험 일정 (중간/기말 추정)
- [ ] 해부학 실습 시작 시점
- [ ] 본과1 OT 일정

## 3. 동산의료원

- [ ] 위치 (의대와의 거리)
- [ ] 견학 동선 (본과1 학생들이 받는 견학)
- [ ] 의국 분위기 (외부 공개 정보 한정)

## 4. 동아리

- [ ] 의대 내 동아리 목록
- [ ] 본과1 가입 가능한 동아리
- [ ] 의예과 동아리 (장윤영이 속할만한)

## 5. 행사

- [ ] 의대 축제 시기 (5~6월 추정)
- [ ] 체전 시기
- [ ] MT 일반적 시기

## 6. 의대 일상 디테일 (분위기용)

- [ ] 해부학 실습 첫날 분위기 (일반적 의대생 경험담)
- [ ] 본과 시험 분위기 (도서관, 족보)
- [ ] 회식 패턴

## 7. 사용 가이드

위 정보를 수집한 후, **게임 시나리오 작성 시 시드(seed)로만 사용**.
계명대를 부정적으로 묘사하지 않는다 (실명 사용이므로).

## 8. 정보 출처 기록

수집한 모든 정보의 출처 URL을 이 파일 하단에 기록.
