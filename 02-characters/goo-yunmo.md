## 2. 성격 / 가치관

- **밝고 쾌활**, 친화력 좋음
- **착하고 봉사정신 투철**, 다정함
- **음란하고 변태 기질** — 단, 외부로 안 드러나는 내면 망상으로만. 행동은 신사적
- **인생 1순위**: 좋은 여성을 만나 행복한 가정을 꾸리는 것
- **이상형**: 착하고 똑똑하고 참한 여성. 외적으로는 "토끼상" — 하얗고 예쁜 토끼 같은
- **연애 자체에 진심**. 가벼운 어프로치보다 진중한 호감 표현 선호
- **돈 씀씀이**: 친구들/연애에 잘 씀. 사치는 아님
- **자기 객관화 능력 있음**: 망상 후 빠르게 자기자각 + 정신 차림
- **소속 동아리 (3개)**:
  - 리온 (RION) — 댄스 동아리
  - 메니스 (MINNIS) — 테니스 동아리, **2027년 회장 예정**
  - 파노라마 — 사진/출사 동아리
- **운동 빈도**: 헬스 주 2~3회 + 테니스 동아리 정기 활동 (운동 캐릭 강조 약함, 동아리 활동으로 대체)

### 4.2 히로인과 카톡 (절제됨, ㅋ 줄어듦)

```
[카톡: 차세린] 오늘 만나서 반가웠어요
[카톡: 구윤모] 저도요
[카톡: 구윤모] 오늘 너무 감사했습니다
[카톡: 구윤모] 다음에 식사라도..?
```

→ 친구한텐 ㅋㅋ 폭격, 호감 가는 사람한텐 절제. 이 갭이 캐릭터 핵심.

### 4.3 오프라인 대화 톤 (대면 대화, **카톡과 분리**)

> **중요**: §4.1·§4.2의 카톡 톤(짧은 단답, ㅇㅇ/ㄱㄱ/ㄷㄷ, ㅋㅋㅋ)은 **메신저 한정**. 대면 대화에서는 더 **부드럽고 예의 있게** — 무뚝뚝/무례 인상 회피.

| 항목 | 카톡 | 오프라인 대화 |
|---|---|---|
| 약자 사용 (ㅇㅇ, ㄱㄱ, ㅇㅋ, ㄴㄴ, ㄷㄷ) | 자유롭게 | **금지** (대신 "응", "어", "아니", "그래") |
| ㅋ/ㅋㅋ 어미 | 친구 사이 자유롭게 | **금지** (대신 `(피식)` 같은 지문) |
| 단답 빈도 | 매우 높음 | 한두 마디 더 붙여 충실히 |
| 어머니/연장자 | 카톡에선 짤막 | **부드럽게 + 가벼운 정중함** ("응, 알겠어", "고마워") |
| 처음 만난 사람 | (카톡 없음) | **존댓말** ("안녕하세요", "잘 부탁드립니다") |
| 동기 친구 | 단답 + ㅋㅋㅋ | 반말 OK, 단 너무 무뚝뚝 단답은 피함 |

**예시 — 어머니와 거실 대화 (오프라인)**:
```
[어머니] 짐 다 챙겼어?
[구윤모] 어. 거의 다 했어.
[어머니] 무리하진 마.
[구윤모] 응. 고마워.
```
→ "어. 거의."처럼 단답으로 끝내지 않음. 한두 마디 덧붙임.

**예시 — 동기 처음 만남 (오프라인)**:
```
[윤하정] 너 과대지?
[구윤모] 응, 맞아. 구윤모.
[구윤모] (살짝 차분히) 앞으로 잘 부탁할게.
```
→ "ㅇㅇ. 구윤모." 같은 카톡 약자는 대면에서 X.

**Why**: 윤모의 실제 성격은 §2 "밝고 쾌활, 친화력 좋음, 다정함". 카톡 톤(분석 기반)을 그대로 대면에 옮기면 이 성격이 안 살아남. 채널 분리로 두 톤을 각각 보존.

### 4.4 모놀로그 톤 (게임 내 핵심)

기본 톤: 가볍지만 솔직, 자기 비하 약간, 망상 빠질 때만 시각적 망상.

```
[구윤모 모놀로그] 와... 진짜 예쁘시다.
[구윤모 모놀로그] 근데 누나는 전공의 2년차시잖아? 의대생이랑 대화할 시간이 있을까.
[구윤모 모놀로그] (망상 시작) 만약에 누나가 야간 당직 끝나고 피곤해서 의국 소파에서 잠들어 있고, 가운이 살짝...
[구윤모 모놀로그] (자기자각) 아 진짜. 정신 차려라 구윤모.
[구윤모 모놀로그] (정상복귀) 진짜로 자고 계셨으면 깨우지 말고 그냥 가야지.
```
