# 02-characters/side-characters.md

> 모든 사이드 캐릭터는 **가공된 이름** 사용 (실명 금지). 카톡 분석상 자주 나오는 친구들이 모티브.

## 1. 동기 친구 그룹 (총 6명, 모두 본과1)

### 1.1 김규민 (가장 친한 친구)
- **자취는 따로 (성서 인근), 베프**. 자주 윤모 자취방에 놀러옴
- **컨셉**: 연애를 매우 하고 싶으나 못 하고 있는 비통한 솔로
- 말투: 약간 까불 + 솔로의 한 ("야 윤모야~ 너만 잘되면 돼?", "ㄱㄱㄱ 솔로의 외침")
- 역할: 윤모의 망상 후 자기자각 트리거 자주 (의심하는 눈빛, 약간의 시기심 포함). 회식·롤·헬스 동행
- 카톡 빈도 최다, 모든 챕터 등장
- 스프라이트 4종: default, smirk, drunk, serious

### 1.2 표경민 (모범생 친구)
- 본과1 성적 1등 후보, 윤하정과 라이벌
- 말투: 차분, 간결, "그래?", "정확히는"
- 역할: 시험 정보 공유, 도서관 동행
- 스프라이트 3종: default, serious, smile

### 1.3 조나단 (분위기 메이커)
- 회식 자리의 분위기 담당
- 말투: 활기, "아 ㅋㅋㅋㅋ", "ㄹㅇㅋㅋ"
- 역할: 회식 시퀀스, MT 시퀀스 핵심
- 스프라이트 3종: default, laugh, drunk

### 1.4 오준혁 (해부조 동기, 여학생, 비공략)
- 윤하정과 같은 해부조의 또 다른 여학생. **이름이 중성적이라 처음엔 남자로 오해받음** (소소한 입체감)
- 말투: 차분, 친근, 윤모/하정 둘 다 친하게 지냄
- 역할: 윤하정의 마음 살짝 윤모에게 알려주는 베프 역할 (트리거)
- 스프라이트 2종: default, smile

### 1.5 이문규 (해부 5조 동기, 회식 톤 메이커)
- 윤하정·오준혁·정욱과 함께 해부 5조. 술자리·축제 부스 분위기 담당
- 말투: 활기, 직설, "야. 누구냐 저분", "윤하정 한 잔 더!", "오늘 풀어진 윤하정 이문규가 처음 봐ㅋㅋ"
- 역할: Ch.2 해부 5조 등장, Ch.5 회식 단톡, Ch.6 H2 부스 운영 + 동성로 회식 자리 톤 메이커
- 스프라이트 1종: default

### 1.6 정욱 (해부 5조 동기, 잡일 도맡는 결)
- 윤하정·오준혁·이문규와 함께 해부 5조. 본과 1학년 동기 안에서 부스 풍선 안내·잡일 도맡는 묵묵한 결
- 말투: 동급생 반말 + 짧은 톤 (이름 부르기, "야"), "풍선 받아왔어", "같이 다는 거 도와줄까?", "와 윤하정 정확하네ㅋㅋ"
- 역할: Ch.2 해부 5조 등장, Ch.6 H2 축제 부스 풍선 담당
- 스프라이트 1종: default

## 2. 교수 / 의국 인물

### 2.1 이태호 교수 (해부학 교수)
- **40대 남성, 영포티**, 쾌활하고 친절. 단 사생활은 약간 문란하다는 학내 소문
- 말투: 친근하면서도 가끔 살짝 도가 넘는 농담, 의학 용어 자유, "자, 다음~"
- 역할: Ch.2 카데바 첫 대면 시퀀스 등장. 권위적이지 않아 학생들이 편하게 대함 → 다만 코미디 포인트로 살짝 위험한 농담 한두 번
- 스프라이트 2종: default, smile

### 2.2 나은영 교수 (생화학 교수, 한설의 지도교수)
- **50대 여성, 미혼, 똑똑하기만 함** (사회성·인간미 약간 부족, 연구실 외 관심사 거의 없음)
- 말투: 학술적·사무적, 표정 변화 적음
- 역할: Ch.2 실험 사고 시 한설을 통해 간접 등장 (직접 등장 X). 한설의 학자 롤모델이자 "이대로 가면 나도 저렇게 될까" 하는 조용한 부담 요인
- 스프라이트 1종: default

### 2.3 남희석 펠로우 (동산병원 내과, 차세린의 선배)
- 30대 후반, 친절
- 역할: Ch.3 견학 가이드, 차세린과 자연스럽게 마주칠 트리거 제공
- 스프라이트 1종: default

### 2.4 승보 (약대 23학번, 김규민 친구, H4 매개)
- **23학번 약대 4학년 / 00년생** (3수 후 23학번 진학). 김규민과 학번 동기·친분. 윤모(04년생 현역) 기준 4살 위 형, 나서윤(03년생) 기준 3살 위 오빠
- 말투: 친근·짧은 톤, 김규민·나서윤·윤모 모두에게 자연스러운 반말 (학번 동기 + 본인이 가장 연장자), 윤모에겐 약간 가벼운 형 톤
- 역할: Ch.4 학식길 합석에서 H4 나서윤을 윤모에게 연결하는 매개. Ch.5 동성로 회식·가창 펜션 MT 합석. Ch.6 H2 단체 자리 끝 합석. Ch.6 H4 도입 모놀로그·대사에서 회상으로 호명
- **화자 라벨 `[약대 동기]` 유지**: Ch.4 첫 등장에서 인물 정체가 시각적으로 일찍 노출되지 않도록 의도적 익명 라벨. 김규민이 첫 대사에서 "어, 승보 형" 호명한 직후부터 모든 호명·지문·모놀로그·단톡에서 "승보"로 일관 호칭 (대사 호명 시 04년생 본과 1학년 동기들은 모두 "승보 형", 의예과 24학번 후배는 "승보 오빠", 03년생 약대 동학번 나서윤도 3살 위라 "승보 오빠"; 지문·내레이션은 호칭 없이 "승보")
- 자산: 스프라이트·avatar 미등록 (화면 최소 등장 매개 인물)
