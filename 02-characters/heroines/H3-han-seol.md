## 2. 외모

- 단발 + 사이드 가르마, 검은 머리
- 흰 실험 가운 + 검은 뿔테 안경 (벗으면 갭)
- 키 162cm, 슬림
- 매력 포인트: 안경 벗었을 때 갭 + 늦은 밤 풀어진 분위기

## 3. 성격

- **차분, 어른스러움**, 말수 적음
- **꼼꼼한 학자형**, 실험 결과에 진지
- **늦은 밤 약점**: 실험실 야간 작업 후 풀어짐
- **착함**: 학생 실수 잘 커버해줌
- **취미**: 클래식 음악, 식물 키우기 (실험실에 작은 화분)

## 4. 말투

- 정중한 존댓말 ("~예요", "~죠")
- 말끝 흐림 ("그러니까... 음...")
- 호감 가는 사람에게는 점차 풀어진 반말 ("응", "그래") — 늦은 밤 한정
- ㅋㅋ/ㅠㅠ 거의 안 씀, 가끔 "후..." 한숨
- 이모지 없음

예시:
```
[한설] 학생, 이 시약 보관 잘못했어요. 다음부턴 주의해주세요.
[한설] (밤 11시, 안경 벗으며) 후... 오늘도 실험 안 끝나네요...
[한설] (살짝 웃으며) 학생도 아직 학교에 있구나. 같은 처지네요.
```

## 5. 첫 만남 (Ch.2)

- **무대**: 생화학실험실, 본과1 첫 실험
- **상황**: 구윤모가 시약 다루다 살짝 사고 (시약병 떨어뜨림)
- **한설의 반응**: 침착하게 정리 + 짧은 주의
- **갭 모먼트**: 실수한 학생 명단에 안 올림 — 조용히 커버
