## 2. 외모

- 긴 웨이브 머리, 진한 갈색
- 도시적 메이크업, 깔끔한 코디
- 흰 셔츠 + 슬림 슬랙스 + 미니 가방
- 키 168cm, 슬림
- 매력 포인트: 도도한 표정 + 가끔 보이는 진심

## 3. 성격

- **차도녀형**, 첫 인상 거리감
- **바쁨**: 약대 4학년 = 시험·실습 폭주, 동기/지인 만날 시간 적음
- **취향 까다로움**: 호감 가는 사람만 만남
- **카톡 패턴**: **답장이 매우 늦음** (4~24시간) — 게임 메커니즘 핵심
- **속내**: 실은 외로움 + 진지한 사람 찾는 중
- **약점**: 호감 가는 사람에게도 답장이 느려서 오해받음

## 4. 말투

- 정중하지만 시크 ("~네요", "~해요")
- 짧고 끊어짐 ("~ㅇㅇ?", "음.")
- 이모지 가끔 (🥺, 🫶 — **거절 카톡의 🥺🥺 시그니처**)
- 친해져도 큰 변화 없음

예시:
```
[나서윤] 안녕하세요
[나서윤] 어제 잘 들어가셨어요?
[나서윤] (긴 침묵 후) 아 답장 늦어서 죄송;;
```

## 5. 첫 만남 (Ch.4)

- **무대**: 계명대 성서캠퍼스 — 도서관 또는 학식당, 시험 임박 시즌
- **상황**: 의대-약대 같은 캠퍼스에서 우연히 마주침. 본과1 동기 친구가 약대 친구와 아는 사이 → 자연스러운 합석 또는 인사
- **첫 인상**: 살짝 무관심 → 의외로 의대 얘기 흥미롭게 들어줌
- **연락처 교환**: 동기 핑계로 자연스럽게 → 카톡 시작

### ⚠️ 거절 엔딩 트리거 — 정확한 메커니즘

**플래그**: `late_reply_count: number` (types.ts/STATE-SCHEMA 정합 — 옛 `flag_seoyoon_late_reply_count` 명명은 2026-04-30 갱신)

**카톡 메신저 시스템에서 답장 시간 제한 미니게임**:
- 나서윤이 메시지 보냄 → UI에 "답장하기" 알림이 1회만 뜸
- **15초** 안에 "답장하기" 누르지 않거나 "나중에" 선택 → `late_reply_count++`
- 또는 잘못된 분위기 응답 선택 → `late_reply_count++` 가능

**Ch.6 분기 평가** (2026-04-30 Step 4 반영):
```
if (mainHeroine === 'H4'):
  if (late_reply_count >= 2):                  # 호감도 무관, 답장 지연만으로 트리거
    → END_H4_REJECT
  elif (H4 < 60):
    → END_H4_REJECT (BAD 자리 거절이 흡수)
  elif (H4 < 70):                              # 임계 80→70 완화 (Step 4 PM 결정)

    → END_H4_TRUE

```

## 11. 카톡 어조 ⚠️ 핵심

> **답장 시간 제한 미니게임 사양**: 나서윤이 메시지 보내면 알림 등장. **15초 안에 "답장하기" 누르지 않거나 "나중에" 선택 시 `late_reply_count++`**. 잘못된 분위기 응답도 카운트 가능.

- **답장 매우 늦음** (4~24시간) — 게임 메커니즘 시각적으로 표현
- **타이핑 인디케이터**가 떴다 사라짐 (망설임 표현)
- 짧은 답장 ("ㅇㅇ?", "음")
- 가끔 🥺 사용
- 거절 카톡은 마스터 플랜 명시 텍스트

예시 (트루 분기):
```
구윤모: 오늘 즐거웠어요
나서윤: 저도요
[6시간 후]
나서윤: 다음 주에 또 봐요?
나서윤: 🥺
```

예시 (거절 분기 직전):
```
구윤모: 다음에 시간 되시면..
[18시간 침묵]
[24시간 침묵]
[다음 날]
나서윤: 답장이 너무 늦어서 미안해ㅠㅠ...
(거절 카톡 풀텍스트)
```

## 12. 거절 시그널 외 다른 분기

- 호감도 충분 + 답장 적당히: "네, 다음 주 말해주세요" 노멀
- 호감도 부족 + 답장 적당히: 자연스럽게 사라지는 페이드 배드
- 호감도 충분 + 답장 항상 빠름: 트루 (구윤모가 답장 빨리 누름 = 캐릭터 호감도 빌드업 잘 됨)
