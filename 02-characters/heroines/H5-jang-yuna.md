## 2. 외모

- 긴 갈색 머리, 자연 웨이브
- 밝은 화장, 또렷한 이목구비
- 캐주얼 코디 (크롭 티 + 청바지 또는 원피스)
- 키 170cm, 슬림 + 키 큰 편
- 매력 포인트: 환한 미소 + 직진 어필의 활기

## 3. 성격

- **직진형**, 호감 표현 적극적
- **활기 만점**, 동아리·행사 적극 참여
- **밝음**, 어두운 분위기 만들지 않음
- **약점**: 가끔 너무 직진해서 분위기 못 읽음
- **취미**: SNS, 카페 투어, 동아리 (의예과 동아리 가입)

## 4. 말투

- "선배~!", "선배님!" 부름이 자주
- "~거든요!", "~잖아요!", "~일까요?"
- ㅋㅋㅋㅋ 자주, 이모지 자주 (😊, 🥹, 🥺, ✨)
- 카톡 답장 매우 빠름 (1~10분)

예시:
```
[장윤영] 선배~ 오늘 식사 같이 하실래요?
[장윤영] 헉 진짜요?? 선배 너무 멋있어요!! ✨
[장윤영] (살짝 토라짐) 선배 요즘 저한테 답장 늦어요 🥺
```

## 5. 첫 만남 (Ch.3)

- **무대**: 동산병원 견학 마치고 학교 복귀, 의대 본관 앞
- **상황**: 동아리 신입 모집 부스 운영 중인 장윤영이 본과1 선배들에게 어필
- **첫 인상**: 너무 적극적이라 살짝 부담 → 점차 매력 발견
- **연락처**: 동아리 가입 안 해도 카톡 주고받기 시작
