## 2. 외모

- 단정한 미디엄 헤어 (어깨선), 검은빛 갈색
- 흰 의사 가운, 안에 셔츠 + 청록 스크럽
- 키 167cm 정도, 마른 편
- 눈매: 차분, 약간 졸려 보일 수 있는 (피로 누적)
- 매력 포인트: 진지하다가 풀어질 때의 갭

## 3. 성격

- **차분, 어른스러움, 학술적 진지함** (기본 톤)
- **번 아웃 직전의 전공의** — 24시간 당직, 피로 누적
- **무방비 순간**: 의국 소파에서 잠깐 잠들거나, 카페인 부족으로 멍할 때 인간적
- **친절하지만 거리 있음** — 의대생을 "꼬마"로 인식
- **좋아하는 음식**: 단 거 (카페인+당분으로 버팀)
- **약점**: 일중독, 자기 돌봄 못 함

## 4. 말투 (CONVENTIONS.md 3.5 기반)

- 기본: 정중한 어른 톤, "~군요", "~네", "~죠"
- 친해지면: 약간 풀어진 반말 ("응", "그래" 정도)
- 학술 모드: 의학 용어 자연스럽게
- 절대 안 쓰는 것: 이모지, ㅋㅋ, ㅎㅎ, 애교

예시:
```
[차세린] 학생, 길 잃었나요? 의국은 이쪽이 아니에요.
[차세린] (피곤한 미소) 후... 오늘 36시간째예요. 이 정도면 환자가 의사 걱정해주는 게 정상이죠.
[차세린] 음, 그 교과서요? 저도 그거 봤어요. 5장이 좀 헷갈리죠?
```

## 5. 첫 만남 (Ch.3)

- **무대**: 동산병원 견학 중 본과1 학생들이 견학 코스에서 길을 잃음
- **상황**: 구윤모가 화장실 다녀오다 의국 복도에서 차세린과 부딪힘
- **차세린의 첫 인상**: 피곤한데 친절. "괜찮아요? 견학생이죠?"
- **구윤모 모놀로그**: 어른스러운 미인 — 1차 망상 페어 발동 가능 지점

## 11. 카톡 어조

- 답장 시간: 일반적으로 1~3시간 (당직 중일 땐 6~12시간)
- 톤: 정중함 유지, 가끔 짧게 ("응", "넵")
- 이모지/ㅋㅋ 안 씀
- 전화/직접 만남 선호 → 긴 대화 카톡으로 안 함

예시:
```
구윤모: 누나 오늘 견학 감사했습니다
차세린: 저야 별거 안 했죠 ㅎ
차세린: 의대생 견학생들 다 귀여워요
구윤모: 다음에 식사라도..?
차세린: ...
차세린: 일정 보고 답장드릴게요
```

(차세린은 ㅎ 가끔 씀 — H4와 차별화 포인트)
