## Claude Code 작업 원칙

1. **마스터 플랜 + CONVENTIONS만 컨텍스트 기본 로드**
2. 작업할 모듈의 .md만 추가 로드
3. 결과는 정해진 .md/코드 파일에만 쓰기
4. 다른 모듈 수정 필요 시: 작업 끝에 "변경 제안" 섹션에 기록만
5. 완료 시 `00-master/PROGRESS-TRACKER.md` 업데이트
6. 마스터 플랜 변경은 `00-master/CHANGELOG.md` 기록 + PM 승인

## ⚠️ 절대 잊으면 안 되는 것

1. **거절 카톡 엔딩 텍스트**는 마스터 플랜 §4.3에 명시된 그대로 사용 (변경 금지)
2. **변태 망상 페어 룰**: (망상 시작) → (자기자각) → (정상복귀) 트리오, 챕터당 1~2회 한도. 자기자각 톤은 욕설 없이 부드럽게(2026-04-28 갱신).
3. **카톡 분석 기반 윤모 말투** (02-characters/goo-yunmo.md §3) 그대로 적용
4. **12세 등급 가드레일** 모든 시각/텍스트에 적용 + **욕설 절대 금지**(2026-04-28 갱신, 모든 등장인물·채널)
5. **사생활/저작권 가드레일** (CONVENTIONS §7): 실명 친구 사용 금지, 카톡 원문 그대로 사용 금지
