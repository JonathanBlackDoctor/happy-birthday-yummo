```
soft anime illustration with K-drama poster aesthetic, painterly skin tones,
gentle pastel palette, cinematic composition, soft volumetric lighting,
detailed environment, emotional storytelling moment,
no text or signage readable, no watermark, no signature
```

## CG 갤러리용 풀버전 (사용자 후처리)

게임 본편: 텍스트박스가 하단 28%를 가려 워터마크 안 보임 → 원본 사용 OK.
**CG 갤러리 풀스크린**: 워터마크 영역 잘라낸 별도 파일 필요.

워크플로:
```
원본 1920×1080 → 하단 130픽셀(워터마크+여유) 잘라냄 → 1920×950
→ 1920×1080 캔버스에 살짝 위로 배치하고 위/아래 여백을 블러 처리 또는 검은색
→ 파일명: cg_serin_first_meet_full.webp
```

→ 자동화 가능: 사용자가 모든 CG 생성 후 일괄 crop 스크립트 실행.
